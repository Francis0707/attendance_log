Welcome to Attendance Log Manager

The Attendance Log Manager is a lightweight and efficient tool designed to manage employee attendance seamlessly. With features like clocking in, clocking out, and viewing attendance logs, it's the perfect solution for small to medium-sized businesses.

Why Use Attendance Log Manager?

  • Simple and easy-to-use interface.
  • Efficient tracking of employee attendance.
  • Cross-platform support for Linux, Windows, and macOS.
  • Reliable local storage using JSON files.

How to Install

  1. Download the appropriate binary for your operating system.
  2. Ensure the binary has execute permissions (for Linux/macOS: chmod +x attendance_log_manager_linux).
  3. Run the binary directly from your terminal or file explorer.

How to Use

  1. Launch the application.
  2. Choose an option from the menu: Clock In, Clock Out, View Logs, or Exit.
  3. Follow the on-screen prompts to manage your attendance logs.
