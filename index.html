Attendance Log Manager

Streamline employee attendance tracking with ease!

Why Use Attendance Log Manager?

The Attendance Log Manager is a simple and effective application designed to help businesses track employee attendance seamlessly. Here are some benefits:

  • Efficiency: Easily track clock-in and clock-out times for employees.
  • Accountability: Keep accurate attendance records to avoid disputes.
  • Automation: Automatically save logs in JSON format for future reference.
  • Cross-Platform: The application works on both Windows and Linux systems.
  • Lightweight: Minimal resource usage with quick setup.

Installation Instructions

Follow these steps to install and use the Attendance Log Manager:

1. Prerequisites

  • Ensure Rust is installed on your system. You can install it by running:
    • curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  • Ensure you have basic command-line knowledge to navigate and run commands.

2. Download the Application

  • Clone or download the project from the repository:
    • git clone https://github.com/your-repo/attendance-log-manager.git
  • Navigate to the project directory:
    • cd attendance-log-manager

3. Build and Run

  • Build the application using the Rust Cargo tool:
    • cargo build
  • Run the application in development mode:
    • cargo run

4. Using the Application

  • When prompted, choose an option from the menu:
    • 1. Clock In: Enter your employee ID to log your clock-in time.
    • 2. Clock Out: Enter your employee ID to log your clock-out time.
    • 3. View Logs: View the attendance history for your employee ID.
    • 4. Exit: Close the application.

Features

The Attendance Log Manager offers:

  • Employee-specific logs stored in JSON format.
  • Automatic creation of a "logs" folder for record keeping.
  • User-friendly menu-driven interface.
  • Error handling for duplicate clock-ins or missing clock-outs.
