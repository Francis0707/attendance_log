Welcome to the Attendance Log Manager

The Attendance Log Manager is a simple and effective solution for managing employee attendance. Whether you run a small business or a large organization, this tool helps you streamline attendance tracking efficiently.

Why Use Attendance Log Manager?

  • Track employee clock-ins and clock-outs effortlessly.
  • Access detailed logs to monitor work hours.
  • Ensure accountability and boost productivity.
  • Store attendance data securely in JSON files.

How to Install and Use

Installation

  1. Ensure you have Rust installed. If not, download it from Rust's official site.
  2. Clone the repository to your local machine: 
    git clone https://github.com/Francis0707/attendance_log.git
  3. Navigate to the project directory: 
    cd attendance_log
  4. Build the project: 
    cargo build
  5. Run the application: 
    cargo run

Usage

  1. Upon running the application, you will see a menu with the following options:
    • Clock In: Enter your employee ID to clock in.
    • Clock Out: Use your employee ID to clock out.
    • View Logs: Check attendance logs for any employee ID.
    • Exit: Exit the application safely.
  2. Logs are automatically saved in the logs directory as JSON files.
  3. To view logs manually, navigate to the logs folder in the project directory.
